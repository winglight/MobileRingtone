Daily Mobile Ringtone | 手机铃声大全

Version 2.2 | Release notes

What's new:

- You could show ringtones already downloaded on SD-card by select "Only Local Music" from the spinner on the right-top of the screen.

- Fix some bugs

v2.1 features:

- You could filter ringtone by different languages by clicking on the Language menu.

- You could search ringtone by name, singer, or other any words by clicking on the Search menu.

- Fix some bugs

v2.0 features:

- Brand new UI for more convenient operation

- Fix some bugs

v1.7 features:

- Add buttons, play, pause, stop, to control playback while playing music

- Add "Remove Ad" function

- Fix some bugs

v1.6 features:

- Add context menu to set the custom ringtone for any contact

- Stop playing ringtone when call in or out

- Improvement of users' experience

v1.5 features:

- Add page down/up function by buttons of volume

v1.4 features:

- Share your favorite ringtones with your friends by MMS, Email, Bluetooth, etc.

v1.2 features:

- Long press on any music link and then select context menu to set the music as the ringtone for call or message.

v1.0 features:

- Long press on E-mail address to open E-mail client

- After downloading an APK file on CheckUpdate page, package installer will be started and install the apk

- New menu item to browse downloaded content

- Auto translate pages into english while local language is english

- Long press on any hyper-link and then select context menu to save this link(including web pages, pictures, APK, etc), or to save the link including next level link(including web pages, pictures)

- Downloaded content prior to be stored in sdcard, of which top folder is /sdcard/download, and content saved from link is put in /sdcard/download/html

最新特性：

- 在屏幕右上角的弹出菜单里，选择“只显示本地铃声”，可以只显示已经下载到SD卡的铃声.

- 修改bug

v2.1 特性

- 在第语言菜单中，可以选择不同语言的铃声.

- 在第查找菜单中，可以根据歌名，歌手，或其他关键字查找铃声.

- 修改bug

v2.0 特性

- 为了更方便您的使用,界面经过了重新设计

- 修改bug

v1.7 特性

- 在播放时，增加一个工具栏可以暂停、播放和停止

- 增加移除广告功能

- 修改bug

v1.6 特性

- 增加菜单，为联系人单独设置铃声

- 电话时自动停止播放铃声

- 改善用户体验

v1.5 特性

- 增加使用音量键翻页的功能

v1.4 特性

- 通过彩信,蓝牙,Email等与朋友分享铃声

v1.2 特性

- 长按任意音乐链接, 在弹出菜单中可以选择设置当前音乐为来电铃声，或者短信铃声

v1.0 特性

- 长按E-mail地址，可以直接打开E-mail客户端发送邮件

- 新增在更新页面下载APK后自动启动安装器安装软件

- 菜单中增加收藏功能,可以查看以往下载的内容

- 增加当本地语言为英语时自动翻译网页内容为英文的功能

- 长按任意链接, 在弹出菜单中可以选择下载链接内容(可以是页面、图片、APK等等格式), 或者下载链接页面及其下一级链接(包括页面、图片内容)

- 下载内容优先保存在SDCARD上, 总目录位置在:/sdcard/download, 其中链接下载内容在/sdcard/download/html
